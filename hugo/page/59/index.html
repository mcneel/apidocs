Namespace: Rhino.Geometry Represents a spacial, Euclidean morph. public class SpaceMorph Properties bool PreserveStructure (summary) True if the morph should be done in a way that preserves the structure of the geometry. In particular, for NURBS objects, True means that only the control points are moved. The PreserveStructure value does not affect the way meshes and points are morphed. The default is false.
