FieldDictionary
Published: 2020-03-11

Namespace: Rhino.Render.Fields

Dictionary containing RenderContent data fields, add fields to this dictionary in your derived RenderContent classes constructor. Get field values using the TryGetdata type methods and set them using the Set() method.

public class FieldDictionary : IEnumerable
