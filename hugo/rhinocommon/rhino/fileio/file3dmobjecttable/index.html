File3dmObjectTable
Published: 2020-03-11

Namespace: Rhino.FileIO

Represents a simple object table for a file that is open externally. This class mimics Rhino.DocObjects.Tables.ObjectTable while providing external eccess to the file.

public class File3dmObjectTable :
    File3dmCommonComponentTable<File3dmObject>,
    IEnumerable<File3dmObject>
