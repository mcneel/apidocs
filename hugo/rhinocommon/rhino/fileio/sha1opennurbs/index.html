SHA1OpenNURBS
Published: 2020-03-11

Namespace: Rhino.FileIO

Provides the OpenNURBS implementation of SHA1.

This class is provided only with the purpose of hashing. It is not meant to be used for any cryptographic purpose.

static byte[] FileSystemPathHash(string path,bool? ignoreCase)
Computes the SHA1 hash of a file system path, converted to UTF8. These file system paths have identical values of FileSystemPathHash():/x/y/z/name.ext\x\y\z\name.ext/x//y//z/name.ext/x/y/a/b/c/../../../z/name.ext/X/Y/Z/NAME.EXT (When ignoreCase is true)
Returns - A 20-byte long SHA1 hash.
