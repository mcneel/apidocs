since 7.0
double MinimumCreaseAngleRadians
When the interior crease option is AtMeshCreases the value of MinimumCreaseAngleRadians determines which coincident input mesh edges generate sub-D creases. If the input mesh has vertex normals, and the angle between vertex normals is > MinimumCreaseAngleRadians at an end of a coincident input mesh edge, the the correspondeing sub-D edge will be a crease.
since 7.0
