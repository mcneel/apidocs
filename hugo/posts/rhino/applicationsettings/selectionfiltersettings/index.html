SelectionFilterSettings
Published: 0001-01-01

Namespace: Rhino.ApplicationSettings

Selection filter settings restrict any selection mode (SelWindow, SelCrossing, SelAll, etc.) to specified object types. Note, selection filter settings are not persistent.

public class SelectionFilterSettings

Properties

static bool Enabled

  • (summary) Enables or disables the global object selection filter.
  • (since) 7.0

static Rhino.DocObjects.ObjectType GlobalGeometryFilter

  • (summary) The global geometry type filter controls which types of geometry will be be filtered. Note, the filter can be a bitwise combination of multiple object types.
  • (since) 7.0

static Rhino.DocObjects.ObjectType OneShotGeometryFilter

  • (summary) The one-shot geometry type filter controls which types of geometry will be be filtered for one selection. Note, the filter can be a bitwise combination of multiple object types.
  • (since) 7.0

  • (summary) Enables or disabled sub-object selection.
