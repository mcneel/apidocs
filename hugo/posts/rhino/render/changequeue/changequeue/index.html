ChangeQueue
Published: 0001-01-01

Namespace: Rhino.Render.ChangeQueue

Base class for ChangeQueue.

Generally used by render plugins to build interactive updating of scenes that are being rendered.

void Flush()

  • (summary) Tell the ChangeQueue to flush all collected events.

    This can trigger a host of Apply* calls.

    The following is the order in which those calls get made if there are changes for that specific data type:

    ApplyViewChange ApplyLinearWorkflowChanges ApplyDynamicObjectTransforms ApplyDynamicLightChanges ApplyRenderSettingsChanges ApplyEnvironmentChanges (background) ApplyEnvironmentChanges (refl) ApplyEnvironmentChanges (sky) ApplySkylightChanges ApplySunChanges ApplyLightChanges ApplyMaterialChanges ApplyMeshChanges ApplyMeshInstanceChanges ApplyGroundPlaneChanges ApplyClippingPlaneChanges

  • (since) 6.0

  • (returns) void This is some return comment
