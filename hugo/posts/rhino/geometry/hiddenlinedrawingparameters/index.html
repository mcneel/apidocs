void SetViewport(RhinoViewport viewport)

  • (summary) Set the viewport for the hidden line drawing (HLD). The viewport supplies the projection that determinest he visibility of curves, and the HLD coordinate system in which the resulting curves are represented. The HLD-coordinate system is a right handed system for 3-D model space, in which the 3rd coordinate direction is the projection direction. In particular the z-coordinate direction points from the scene toward the camera.
  • (since) 6.0
