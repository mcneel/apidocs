Quaternion
Published: 0001-01-01

Namespace: Rhino.Geometry

Represents the four coefficient values in a quaternion. The first value a is the real part, while the rest multipies i, j and k, that are imaginary.quaternion = a + bi + cj + dk

static -

  • (summary) Subtracts a quaternion from another one. This computes the difference of each quaternion coefficient with its correspondant and returns a new result quaternion.
  • (since) 5.0

  • (since) 5.0

Quaternion Inverse

  • (summary) Computes a new inverted quaternion, (a/L2, -b/L2, -c/L2, -d/L2),where L2 = length squared = (aa + bb + cc + dd). This is the multiplicative inverse, i.e., (a,b,c,d)*(a/L2, -b/L2, -c/L2, -d/L2) = (1,0,0,0). If this is the zero quaternion, then the zero quaternion is returned.
  • (since) 5.0
