Box
Published: 0001-01-01

Namespace: Rhino.Geometry

Represents the value of a plane and three intervals in an orthogonal, oriented box that is not necessarily parallel to the world Y, X, Z axes.

Constructors

Box(BoundingBox bbox)

  • (summary) Initializes a new Box that mimics a BoundingBox struct. The orientation plane of the Box is coincident with the World XY plane.
  • (since) 5.0

  • (summary) Initializes a world aligned box from a base plane and a boundingbox.
  • (since) 5.0

Box(Plane basePlane,GeometryBase geometry)

  • (summary) Initializes a box that contains a generic piece of geometry. This box will be aligned with an arbitrary plane.
  • (since) 5.0
