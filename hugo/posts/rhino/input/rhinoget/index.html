RhinoGet
Published: 0001-01-01

Namespace: Rhino.Input

Base class for GetObject, GetPoint, GetSphere, etc.

You will never directly create a RhinoGet but you will use its member functions after calling GetObject::GetObjects(), GetPoint::GetPoint(), and so on.

Provides tools to set command prompt, set command options, and specify if the “get” can optionally accept numbers, nothing (pressing enter), and undo.
