# Events
## static UndoRedoChanged
- (summary) 
     Called after undo or redo has occurred for document settings.
     
- (since) 6.0
## static UndoRedoEndedChanged
- (summary) 
     This event is raised when undo/redo ends in rdk.
     
- (since) 7.0
